Listas Lineales Enlazadas

Teoría Básica

¿Qué es una Lista Lineal Enlazada?

Una lista lineal enlazada es una estructura de datos compuesta por nodos, donde cada nodo contiene un dato y una referencia al siguiente nodo. A diferencia de los arreglos, los nodos no están en posiciones de memoria contiguas.

Analogía del Mundo Real

Imagina una cadena de eslabones: cada eslabón (nodo) está unido al siguiente mediante un gancho (referencia). No necesitas conocer la posición exacta de cada eslabón, solo seguir la cadena desde el inicio.

📦

Nodo (Unidad básica)

Cada nodo almacena un dato y la referencia al siguiente. Si no hay siguiente, apunta a null.

🔗

Enlace dinámico

El tamaño crece o decrece en tiempo de ejecución, sin necesidad de declarar un tamaño fijo.

👆

Referencia cabeza

La lista siempre recuerda dónde está el primer nodo (la cabeza). Desde ahí se puede recorrer todo.

¿Por qué usar listas en lugar de arreglos?

Los arreglos tienen tamaño fijo y las inserciones/eliminaciones en el medio son costosas (hay que desplazar elementos). Las listas enlazadas resuelven ambos problemas a cambio de no poder acceder directamente a la posición i.

Tamaño dinámico Inserción O(1) al inicio Sin desplazamientos Acceso secuencial
Implementación Java

La Clase Nodo

El Nodo es la unidad fundamental de una lista enlazada. Contiene exactamente dos atributos: el dato que almacena y la referencia al siguiente nodo.

// Clase Nodo — unidad básica de la lista class Nodo { // Atributo 1: el dato entero almacenado int dato; // Atributo 2: referencia al siguiente nodo Nodo siguiente; // Constructor: crea un nodo con dato, // el siguiente empieza apuntando a null Nodo(int dato) { this.dato = dato; this.siguiente = null; } }

¿Por qué siguiente es de tipo Nodo?

Porque un nodo apunta a otro nodo del mismo tipo. Esta auto-referencia es lo que permite encadenar nodos uno tras otro, formando la lista.

Implementación Java

La Clase ListaLineal

La clase ListaLineal gestiona la colección de nodos. Su único atributo es cabeza, la referencia al primer nodo. Si cabeza == null, la lista está vacía.

class ListaLineal { // Solo necesitamos conocer el primer nodo Nodo cabeza; // Constructor: lista vacía ListaLineal() { this.cabeza = null; } void insertarInicio(int dato) { Nodo nuevo = new Nodo(dato); nuevo.siguiente = this.cabeza; this.cabeza = nuevo; } void insertarFinal(int dato) { Nodo nuevo = new Nodo(dato); if (this.cabeza == null) { this.cabeza = nuevo; return; } Nodo temp = this.cabeza; while (temp.siguiente != null) temp = temp.siguiente; temp.siguiente = nuevo; } void eliminarInicio() { if (this.cabeza != null) this.cabeza = this.cabeza.siguiente; } void eliminarFinal() { if (this.cabeza == null) return; if (this.cabeza.siguiente == null) { this.cabeza = null; return; } Nodo temp = this.cabeza; while (temp.siguiente.siguiente != null) temp = temp.siguiente; temp.siguiente = null; } }

La cabeza siempre cambia al insertar al inicio

El nuevo nodo apunta a quien era la cabeza anterior, y luego se convierte en la nueva cabeza. Solo movemos referencias, sin copiar datos.

Operaciones

Operaciones Fundamentales

Una lista lineal define cuatro operaciones principales para insertar y eliminar nodos:

insertarInicio(dato)

Crea un nuevo nodo y lo coloca al frente. El nuevo nodo apunta a la antigua cabeza, y la cabeza pasa a ser el nuevo nodo.

Complejidad: O(1)

insertarFinal(dato)

Recorre la lista hasta encontrar el último nodo (cuyo siguiente == null) y lo enlaza al nuevo nodo creado.

Complejidad: O(n)

eliminarInicio()

Mueve la cabeza al siguiente nodo. El nodo que era cabeza queda desenlazado y el recolector de basura de Java lo elimina.

Complejidad: O(1)

eliminarFinal()

Recorre la lista hasta el penúltimo nodo y pone su siguiente = null, desconectando al último nodo.

Complejidad: O(n)
